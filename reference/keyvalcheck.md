---
title: keyvalcheck
---
```
 KEYVALCHECK is a helper function for parsing optional key-value input pairs.

 Use as
   keyvalcheck(argin, 'required',  {'key1', 'key2', ...})
   keyvalcheck(argin, 'forbidden', {'key1', 'key2', ...})
   keyvalcheck(argin, 'optional',  {'key1', 'key2', ...})

 See also KEYVAL
```
